# Munamii Cakery

A responsive React website built for **Munamii Cakery**, showcasing cakes and desserts with a clean, modern design.  
Built using **React** and **Bootstrap**.

---

## Components
- **Navbar** – top navigation bar  
- **Hero** – intro section with headline and call-to-action  
- **ProductGrid** – displays cake items using data from JSON  
- **Footer** – contact info and social links  

---

## Tech Stack
- React (Create React App)  
- Bootstrap 5 (CDN)  
- CSS3 / Flexbox  
- JavaScript (ES6)
